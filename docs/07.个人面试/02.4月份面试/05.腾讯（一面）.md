- 一个数组，数据出现偶数次，其他的是奇数，怎么找出来奇数

  哈哈哈，还好我当时背了。。。

  因为对于任意一个数 k，k^k = 0，k^0 = k，所以将array中所有元素进行异或，那么个数为偶数的元素异或后都变成了0，只留下了个数为奇数的那个元素。
